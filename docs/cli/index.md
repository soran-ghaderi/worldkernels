# CLI Reference

GPU-first world model simulation engine

## Usage

```bash
worldkernels <command> [options]
```

## Commands

| Command | Description |
|---------|-------------|
| [`bench`](bench.md) | Benchmarking CLI commands |
| [`serve`](serve.md) | Start HTTP/WebSocket server |
| [`version`](version.md) | Show version |

## Global Options

| Flag | Description |
|------|-------------|
| `--help`, `-h` | Show help message and exit |
| `--version`, `-V` | Show version and exit |
