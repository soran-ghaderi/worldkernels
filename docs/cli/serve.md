# <code class="doc-symbol doc-symbol-heading doc-symbol-command"></code> worldkernels serve

Start HTTP/WebSocket server

## Usage

```bash
worldkernels serve [--host HOST] [--port PORT]
```

## Options

| Flag | Metavar | Description |
|------|---------|-------------|
| `--host` | `HOST` | — |
| `--port` | `PORT` | — |

## Examples

```bash
worldkernels serve --host 0.0.0.0 --port 8000
```
